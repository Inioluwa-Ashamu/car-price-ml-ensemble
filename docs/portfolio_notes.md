# Portfolio Notes

## Recruiter Framing

This project demonstrates practical tabular machine learning: data cleaning, feature engineering, ensemble modelling, dimensionality reduction, and model explainability.

## Best Talking Points

- Built a full regression workflow from raw advert data to model comparison.
- Used tree ensembles and stacking to improve validation performance.
- Added SHAP and partial dependence plots to explain model behaviour.
- Compared predictive models against PCA, Isomap, polynomial, and clustering-based alternatives.
